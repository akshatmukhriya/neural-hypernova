## ⚠️ Prerequisites & Constraints

### 1. Hardware Requirements (Local Mode)
The "Neural Supernova" simulation runs on Minikube.
- **RAM:** Minimum **8GB** System RAM (Docker must be allowed to use at least **6GB**).
- **Disk:** 20GB Free.
- **CPU:** 4 Cores.

**Critical Config for Mac Users:**
Ensure your Docker Desktop resource limit is set to **at least 6GB**.
*Settings -> Resources -> Memory -> Slide to 6GB+*

### 2. Software Dependencies
- Docker Desktop (Must be running)
- Python 3.9+
- The "Jockey" (GitHub Runner) handles the rest.
